mongodb的命令

2016-07-31
mongodb

文章目录

    • Object
      • Object.bsonsize(<doc>)

本文主要汇总mongodb的命令行。

系统：CentOS7 + docker 1.6

mongodb版本： 3.0.2

mongo shell版本：3.0.4

Object

Object.bsonsize(<doc>)

这个命令返回文档的bson的大小，单位是字节。

例子：

db.test.status
Object.bsonsize(db.test.findOne({bar2:"baz"}))

可以看下是不是跟总和一致。这里一定要使用findOne，find返回的是CURSOR的size，而不是当前文档的。
