OpenWrt编译系统变量整理

2017-10-07
openwrt

文章目录

    • 编译系统的路径相关变量说明
    • 编译相关的变量说明

本文是针对在Makefile中会经常遇到的一些变量的说明，对于我们自己写Makefile或者理解别的package的Makefile都会有帮助。

下文所描述的大部分的变量，我们都可以在文件rootdir/rules.mk以及rootdir/include/*.mk中找到定义，以及对这些变量的赋值。

编译系统的路径相关变量说明

变量名 变量说明 备注
PKG_BUILD_DIR 工程编译目录通常指向工程的根目录 无
STAGING_DIR op的staging_dir目录中的当前target的目录 无
STAGING_DIR_ROOT op的staging_dir目录 无
LINUX_DIR 通常指向linux内核所在目录 无
INCLUDE_DIR op根目录的include目录 无
SCRIPT_DIR op根目录下的script目录 无
STAGING_DIR_HOST op根目录下的staging_dir下的host目录 无
TOOLCHAIN_DIR op根目录下的staging_dir下的当前工具链目录 无
ROOTDIR 当前系统的根目录(/) 无
TOPDIR 当前openwrt工程的根目录 无
BUILD_DIR_HOST 工程编译目录下的host目录 无

编译相关的变量说明

变量名 变量说明 备注
CMAKE_OPTIONS 添加cmake的define 无
CMAKE_HOST_OPTIONS 添加cmake的host build的define 无
PKG_BUILD_DEPENDS 依赖包不会被选择，但是编译工程当前工程前必须先编译此工程 无
HOST_BUILD_DEPENDS 如果执行host编译，那么这个变量可以配置host编译说依赖的工程 无
